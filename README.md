# Prompt Vault

This is the code for an assignment for the Secure Computing module in the Cybersecurity MSC in UCD.

# Requirements

Use Spring Boot

Does not require the use of an AI API. AI responses can be mocked to read:

`This is a simulated AI response.`

## Login and logut functionality

Users:
* Admin - can view the list of registered Users. Can enable and disable regular users.
* User - preregistered has username, email, role, account status.
* Categories - Admin can add, edit and delete prompt categories. Categories have name and description. 
Example categories:
* Coding
* Research
* Cybersecurity
* Legal
* HR
* Personal productivity
Policy Keywords
